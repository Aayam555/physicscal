Problem 2

Consider an infinite network consisting of resistors (resistance of each of them is $r$) shown in the figure below. Find the resultant resistance $R_{AB}$ between points A and B.

Solution

This problem can be solved by recognizing the self-similar nature of the infinite network. If we remove the first section of the network, the remaining part is still an identical infinite network with the same resultant resistance, $R_{AB}$.

We can represent the circuit as a combination of the first section and the equivalent resistance of the rest of the network, which we call $R_{eq}$.

From the diagram, we can see that the equivalent resistance $R_{eq}$ is the series combination of the first horizontal resistor with the parallel combination of the vertical resistor and the equivalent resistance of the rest of the network. This gives us the equation:

$$R_{eq} = r + \frac{1}{\frac{1}{r} + \frac{1}{R_{eq}}}$$

$$R_{eq} = r + \frac{r \cdot R_{eq}}{r + R_{eq}}$$

Now, we solve this equation for $R_{eq}$. We multiply both sides by $(r + R_{eq})$:

$$R_{eq}(r + R_{eq}) = r(r + R_{eq}) + r R_{eq}$$

$$r R_{eq} + R_{eq}^2 = r^2 + r R_{eq} + r R_{eq}$$

$$R_{eq}^2 - r R_{eq} - r^2 = 0$$

This is a quadratic equation. We can solve for $R_{eq}$ using the quadratic formula, where $a=1$, $b=-r$, and $c=-r^2$.

$$R_{eq} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} = \frac{-(-r) \pm \sqrt{(-r)^2 - 4(1)(-r^2)}}{2(1)}$$

$$R_{eq} = \frac{r \pm \sqrt{r^2 + 4r^2}}{2} = \frac{r \pm \sqrt{5r^2}}{2}$$

$$R_{eq} = \frac{r(1 \pm \sqrt{5})}{2}$$

Since resistance must be a positive value, we take the positive root.

The resultant resistance is $R_{AB} = r \left( \frac{1 + \sqrt{5}}{2} \right)$.

Note that the term $\frac{1 + \sqrt{5}}{2}$ is the golden ratio, often denoted by $\phi$.
